# Currency Exchange Tracker

Сервис отслеживает курс USD/KZT и отправляет уведомления при его изменении. Реализован на Python c асинхронным поллингом публичного API exchangerate.host (без сторонних зависимостей).

## Возможности
- Периодический запрос курса USD/KZT (или другой пары) через exchangerate.host.
- Отправка уведомлений при первом получении курса и при каждом изменении больше заданного порога.
- Простые уведомления в консоль (можно заменить на любой другой `Notifier`).

## Установка
```
pip install -r requirements.txt
```

## Запуск
По умолчанию сервис опрашивает курс каждую минуту и уведомляет при изменении больше 0.01:
```
python -m currency_tracker.cli
```

Полезные флаги:
- `--interval` — интервал опроса в секундах (по умолчанию 60).
- `--tolerance` — минимальное изменение курса для уведомления (по умолчанию 0.01).
- `--base` и `--quote` — коды валют (по умолчанию USD/KZT).

Остановить сервис можно комбинацией `Ctrl+C` или отправкой SIGTERM.
